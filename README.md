# The Now Ghost Client Portal

Simple Node app for:

- admin login
- per-client switching
- pasting posts into a queue
- monthly calendar view
- client comments
- post approval workflow

## Run locally

```powershell
cd "C:\Users\DannyGleason\Documents\New project\the-now-ghost-client-portal"
node server.js
```

Open:

`http://localhost:3030`
